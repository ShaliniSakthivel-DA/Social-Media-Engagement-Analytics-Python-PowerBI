# Social-Media-Engagement-Analytics-Python-PowerBI
Social Media Engagement Analytics project analyzes social media data including posts, users, likes, comments, shares, impressions, watch time, followers, and engagement rate. It focuses on data cleaning, EDA, visualization, and insights to understand content performance, audience behavior, and improve content strategies to improve content reach!!!
# Social Media Engagement Analytics

## Project Overview

This project analyzes a social media engagement dataset containing information about posts, users, and engagement metrics such as likes, comments, shares, impressions, watch time, followers, and engagement rate.

The main objective of this project is to clean the data, perform exploratory data analysis, visualize trends, and generate insights about content performance and user behavior.

---

## Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI

---

## Dataset Description

The dataset contains information about:

- Post type (Image, Reel, Text, Video)
- Content category
- User details
- Likes
- Comments
- Shares
- Impressions
- Followers
- Watch time
- Engagement rate
- Sentiment

---

# Project Workflow

## 1. Data Cleaning

Performed data preprocessing:

- Checked missing values
- Handled missing values using:
  - Mode imputation
  - Median imputation
- Removed duplicate records
- Corrected data types
- Standardized categorical columns like:
  - Gender
  - Sentiment

---

## 2. Feature Engineering

Created new features:

### Hashtag Count

Extracted the number of hashtags used in each post.

### Engagement Score

Calculated engagement score using:

- Likes
- Comments
- Shares

---

# Exploratory Data Analysis

## Descriptive Analysis

Analyzed:

- Distribution of post types
- Content category performance
- Engagement patterns
- User behavior

### Key Findings:

- Reel posts have the highest number of posts.
- Food category content received the highest average likes.
- Sentiment categories showed similar follower distributions.
- Likes, comments, and shares increase with impressions.

---

# Diagnostic Analysis

Used:

- Correlation analysis
- GroupBy analysis
- Data visualization

### Insights:

High-performing posts depend on:

- Content category
- Post type
- Audience interest
- Reach and impressions

# Predictive Analysis

Based on observed trends:

- High-quality reels can generate better engagement.
- Popular categories like food content attract more audience.
- Increasing impressions improves interactions.
- Hashtags improve content discoverability.
- Engagement rate helps identify future performing posts.
